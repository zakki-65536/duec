# データベースの作成手順
## インテリ、情シスそれぞれの科目一覧のExcelファイルを作成
1. 履修要項のpdfファイルの科目一覧ページを抽出し、Excelに変換
2. 学科区分(インテリ/情シス/両方)、配当年次、科目コード、科目名、単位数をまとめたExcelファイルを作成
3. シラバスのURLの規則性を調べ、各科目のURLをExcelファイルに追加
> [!IMPORTANT]
> 成績評価結果を取得するため、シラバスは2024年度のものを参照するようにしてください。

## 複数のURLをまとめて開く
1. URLを改行区切りでコピー
2. Chrome のブックマークバーで右クリック →「ページを追加」
3. 名前：`Bulk Open`
4. URL に以下を貼り付け：
```
javascript:(function(){var urls=prompt("開きたいURLを改行区切りで入力してください");urls.split(/\n/).forEach(u=>{if(u.trim())window.open(u.trim(), "_blank");});})();
```
5. 保存
6. ブックマークをクリックして、すべてのURLを貼り付け
> [!NOTE]
> ポップアップがブロックされた場合は、ブロックを解除してください。

## すべてのページのHTMLファイルを保存
1. 各シラバスのページを開き、`Ctrl+S`
2. インテリと情シスでフォルダを分けてHTMLファイルを保存

## HTMLファイルから情報を取得するプログラムを作成
概要のタイトルを示す以下の`p`タグの次の`p`タグは、概要の文章です。
```
<p><font color="#229072"><b>＜概要/Course Content Summary＞</b></font></p>
```
到達目標のタイトルを示す以下の`p`タグの次の`p`タグは、到達目標の文章です。
```
<p><font color="#229072"><b>＜到達目標/Goals,Aims＞</b></font></p>
```
開発者ツールでHTMLを見ることでこのような規則を発見し、HTML構造をもとに各項目の文章を取得するプログラムを作成してください。
